Tugas 2 (Pipelines)#

Import Database#

Data yang digunakan adalah data Iris dengan nama kolom sepallength, sepal_w, petallength, petalwidth.

Database yang digunakan antara lain:

  1. PostgreSQL

  2. Elephant

  3. MySQL

  4. SQL Server

Akses Database#

Untuk mengakses semua database dapat menggunakan ngrok. di ngrok di setting TCP port dari masing-masing database. Agar ngrok bisa berjalan bersamaan, maka di ngrok.yml di tambahkan sebuah tunnel yang mewakili port database. Untuk codenya ada dibawah ini:

# This is formatted as code
tunnels:
  mysql:
    addr: 3306
    proto: tcp
  postgre:
    addr: 5432
    proto: tcp

Setelah itu mengakses databasenya sesuai dengan tcp yang muncul di ngrok. yang diganti adalah host dan portnya

Get Data From DB#

  1. PostgreSQL

Untuk mengakses database PostgreSQL local menggunakan ngrok. kemudiian diquery untuk mendapatkan kolom yang ada didatabase PostgreSQL. kolom yang diambil merupakan kolom sepallength dari tabel iris. library yang digunakan untuk mengakses database ini yaitu import psycopg2.

  1. Elephant

Untuk mengakses database Elephant yang ada di postgreSQL, menggunakan library import psycopg2. kemudian didalam library tersebut ada parameter yang harus dimasukkan. data yang diambil dari kolom sepal_w dari tabel iris.

  1. MySQL

Untuk mengakses database MySQL menggunakan library mysql-connector. agar bisa diakses publik, dapat menggunakan ngrok. didalam parameter mysql.connector disetting host dan port sesuai ngrok yang disediakan. data yang diambil dari kolom petallength dari tabel iris.

  1. CSV

Karena SQL Server tidak dapat diakses oleh orchest, maka diganti dengan data CSV dengan kolom petalwidth dan class. untuk mengakses data CSV tersebut menggunakan library pandas.

Merge Data#

Untuk menggabungkan dataframe, disini menggunakan join dengan bantuan library pandas.

Load Model#

Model yang digunakan adalah KNN dan Naive Bayes Gaussian.

Hasil Akurasi#

Hasil Akurasi KNN = 0.9666666

Hasil Akurasi Naive Bayes = 1
